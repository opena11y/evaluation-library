OpenA11y Evaluation Library

Welcome to the OpenA11y Evaluation Library

The purpose of the OpenA11y Evaluation libary is to support accessible design for implementation of W3C Web Content Accessibility Guidelines level A and AA requirements by creating rules that are based on the following W3C documents:

  • W3C WAI Web Content Accessibility Guidelines 2.0 (WCAG 2.0).
  • W3C WAI Web Content Accessibility Guidelines 2.1 (WCAG 2.1).
  • W3C WAI Web Content Accessibility Guidelines 2.2 (WCAG 2.2).
  • W3C WAI Accessible Rich Internet Application (ARIA) 1.2 specification.
  • W3C WAI Accessible Rich Internet Application (ARIA) 1.3 (Working Draft 23 January 2024).
  • W3C ARIA in HTML specification.
  • W3C ARIA Authoring Practices Guide
  • W3C Accessible Name and Description Computation
