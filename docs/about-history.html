History

The Evaluation library was originally developed as part of the accessibility working group of OpenAjax Alliance, when the OpenAjax Alliance ceased operations the accessibility evaluation library was rebranded to the OpenA11y GitHub organization.

View the release notes for the evaluation library.

Mission

  • To support the understanding and use of the following W3C specializations:
    • Accessible Rich Internet Application (ARIA) 1.2 Specification
    • Accessible Name and Description Computation 1.1
    • ARIA in HTML Specification
    • Web content Accessibility Guidelines 2.0
    • Web content Accessibility Guidelines 2.1
    • Web content Accessibility Guidelines 2.2
  • Open Source for anyone to freely use in their own projects.
  • Support the best practices for implementing accessible online resources.
