Rule: Color contrast of graphics

Rule ID Color 5
Conformance Required
Definition Color contrast of non-text content in graphical objects.
Purpose
  • For people with color blindness and other forms of visual impairments will not be able to see colors or color differences in the grphic.
Conformance Required
WCAG Success Criteria

Success Criterion 1.4.11 Non-text Contrast

Other Related Success Criterion:

Rule Category Color/Content
Rule Scope element
Techniques
  • Identify each graphic on the page that includes information required for understanding the content (i.e. excluding graphics which have visible text for the same information, or are decorative).
  • Check the contrast of the graphical object against its adjacent colors
  • If there are multiple colors and/or a gradient, choose the least contrasting area to test
  • If the least-contrasting area is less than 3:1, assume that area is invisible, and determine if the graphical object still understandable.
